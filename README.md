## DMIT 是什么，WEE 系列又是什么定位

## WEE 套餐核心参数拆解

DMIT 的套餐命名有自己的逻辑，WEE 是入门档，往上还有 TINY、Small、Medium 等规格。不同地区节点的 WEE 套餐配置略有差异，但基本框架是一致的：

- **vCPU**：1 核
- **内存**：768MB 或 1GB（视节点）
- **存储**：10GB SD
- **流量**：月流量限额（双向计算，注意）
- **网络**：精品线路接入（CN2 GIA / CMIN2 / 软银，视节点而定）
- **IP**：1 个 IPv4 + IPv6

流量双向计算这点很多人没注意，上传和下载都算进去。如果你跑的是高流量业务，要提前算好。

## WEE 和更高档套餐怎么选

很多人纠结 WEE 够不够用，这里给个简单判断框架：

**选 WEE 的理由：**
1. 预算在 $10–$15/月 以内
2. 用途是轻量建站、代理节点、个人工具
3. 流量消耗不大（月均 100GB以内）
4. 只需要一个稳定的低延迟 IP

**应该往上升级的信号：**
1. 跑 Docker 多容器，内存经常吃满
2. 月流量超出 WEE 限额
3. 需要更高的突发带宽
4. 业务对 CPU 有持续性需求

WEE 的定位很清晰——它不是给你跑生产环境的，是给你用来「够用就好」的那种场景。我自己就是这么用的，跑了快两年没出过什么大问题。

## FAQ

### DMIT WEE 套餐支持退款吗？

DMIT 提供 72 小时内退款保证，新购套餐如果不满意可以在 72 小时内申请退款。超过这个时间窗口就不支持了，所以买之前最好先确认节点库存和自己的需求匹配。

### WEE 套餐的流量是怎么计算的？

双向计算，也就是上传和下载都算进月流量配额里。香港 WEE 是 250GB/月，日本 WEE 是 200GB/月，洛杉矶 WEE 是 1TB/月。超出后会限速而不是直接断网，但限速后基本不可用。

### DMIT 香港节点对大陆访问速度怎么样？

香港 WEE 走的是 CN2 GIA 线路，这是目前对大陆访问质量最好的商业线路之一。实测延迟在 30–50ms 区间，晚高峰波动比普通 BGP 线路小很多。如果你在意大陆到香港的访问质量，这个节点是 WEE 系列里最值得买的。

### WEE 套餐可以安装什么系统？

支持主流 Linux 发行版，包括 Debian、Ubuntu、CentOS 等，可以通过后台自助重装。不支持 Windows。

### DMIT 经常缺货怎么办？

这是真实情况，热门节点（尤其是香港）确实会售罄。建议直接进官网查看实时库存，或者关注他们的官方 Twitter/X 账号，补货时会有通知。

### 年付和月付哪个划算？

以洛杉矶 WEE 为例，月付 $6.90 × 12 = $82.8，年付 $49.90省了将近 $33。如果你确定会长期用，年付明显更合算。香港和日本节点目前没有看到年付选项，只有月付。
